Overview

The visualization explores differences in college affordability across institutions using geographic and institutional data.

Data
Each institution inclues:

Affordability gap (dollar values)
Pell Grant participation rate
Locale type (urban, suburban, town, rural)
BEA economic region
Carnegie Classification
Geographic coordinates
Implementation
HTML, CSS, JavaScript
Leaflet.js for mapping
Hosted on GitHub Pages

Features
Filtering by region, institution type, and demographic indicators
Color-coded affordability gap visualization
Search functionality for institutions
Clickable markers with detailed information panels
Purpose

This project is designed to help visualize and compare differences in college affordability across the United States.
